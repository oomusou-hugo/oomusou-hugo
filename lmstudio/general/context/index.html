設定 Session 的 Context Length

2026-07-11

LM Studio 可根據需求設定單一 Session 最大的 Context Length。

Version

LM Studio 0.4.19

Context Length

  • 選擇右側倒數第二個的 My Models 列出已下載到本機的 model，選擇要設定的 model
  • 在 Load 下的 Context Length 設定 16384 (16K)

# LM Studio

  • 設定 Model 的 System Prompt
  • 刪除已下載 Model
