LM Studio 初體驗

2026-07-11

LM Studio 提供 GUI 方式管裡 AI Model，從 聊天、啟動 Model、管理 Model、下載 Model，都可以在 LM Studio 中實現。

Version

LM Studio 0.4.19

LM Studio

  • 到 官網 下載 mac 版的 LM Studio

Download Model

  • 右側最下方的 Model Search 可下載 model

My Models

  • 右側倒數第二個的 My Models 可列出已下載到本機的 model

Load Model

  • 右側第二個的 Developer 可列出 已載入 的 model

  • 選擇要載入的 model

Chat

  • 右側第一個的 Chat 可測試 model 是否載入成功

  • 選擇已經載入的 model

  • 輸入 prompt 測試 model 可否正常回應

Unload Model

  • 右側第二個的 Developer 可列出 已載入 的 model
  • 按 Eject 可卸載 已載入 的 model

# LM Studio

  • 刪除已下載 Model
  • Claude 自動做到底不詢問
