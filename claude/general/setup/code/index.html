Claude Code 初體驗

2026-05-03

以 CLI 方式使用 Claude Code。

Version

Claude Code 2.1.118

Homebrew

$ brew install --cask claude-code
  • 使用 Homebrew 安裝 Claude Code

Claude Code

$ claude
  • 進入專案目錄，輸入 claude 執行 Claude Code

Theme

  • 選擇 Dark mode

Claude Account

  • 選擇 Claude account with subscription

  • Continue with Google：以 Google ID 登入

  • 登入 Claude 帳號成功

Trust Folder

  • Yes, I trust this folder：讓 Claude Code 管理此專案目錄

Welcome

  • Claude Code 設定成功

Conclusion

  • CLI 為 Claude Code 的基礎，可在此學到 Claude Code 核心技術
