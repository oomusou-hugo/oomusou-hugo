如何關閉 Safari 的 Cache ?

2019-06-26

若要使用 Safari 為 Web 開發主力，首先遇到就是 Safari 預設將 Cache 開啟，造成開發上不便，該如何關閉 Cache 呢 ?

Version

macOS Mojave 10.14.5
Safari 12.1.1

Web Inspector

⌘ ⌥ I 開啟 Web Inspector，選擇 Network tab，按下右側 disable cache icon。

Conclusion

  • Safari 舊版本是在 Develop menu 下的 Disable Caches，但現在已經移到 Web Inspector 內
