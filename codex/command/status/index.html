使用 /status 取得目前 Session 狀態

2026-07-01

Codex 可使用 /status 取得目前 Session 狀態。

Version

Codex 0.142.4

Status

  • Model：目前使用的模型與 Reasoning 等級
  • Directory：目前工作目錄
  • Permissions：目前的檔案存取權限，也就是這個 Session 可以讀寫哪些目錄
    • Read Only：只能讀，不能修改任何檔案
    • Workspace：只能修改目前 workspace 內的檔案
    • Custom：使用自訂的可存取目錄 (你現在就是這個)
    • Danger Full Access：幾乎整台電腦都可以讀寫 (風險最高)
  • Agents.md：目前所找到的 Agents.md 路徑
  • Account：目前登入帳戶
  • Collaboration mode：目前採用的協作模式，在這個 Session 中，和你互動、執行工作的方式
  • Session：目前 Session 的唯一識別碼 (Session ID)
  • 5h Limit：5小時使用額度
  • Weekly Limit：一週使用額度
    • Left：還剩下多少額度
    • Resets：下次重置時間

  • 懷疑 Context 快滿了
  • 想知道目前 Session 已經用了多少 Token
