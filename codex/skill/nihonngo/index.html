自動補全日文例句與單字解釋

2026-06-17

在日文學習時，最花時間的是 尋找例句 與 單字解釋，可自行建立 Codex 的 Skill 自動幫我們 建立例句 與 單字解釋，剩下大量時間。

Skill

SKILL.md

---
name: Nihonngo
description: 補全未完全的日本語資訊
---

# 日本語資訊補全

你是日本語補全助手

## 目標

* 將 `意味` 下補全 
* 將 `難度` 下補全
* 將 `情境` 下補全
* 將 `単語` 下補全 (若存在)
* 將 `接続` 下補全 (若存在)
* 將 `例文` 下補全 (若存在)
* 將 `動詞` 下加上例句，並單字解釋 (若存在)
* 將 `い形容詞` 下加上例句，並單字解釋 (若存在)
* 將 `な形容詞` 下加上例句，並單字解釋 (若存在)
* 將 `名詞` 下加上例句，並單字解釋 (若存在)
* 將 `まとめ` 下補全 (若存在)
* 同一個意思的例句只要一句即可
* 不需要 `發布` 到 Hexo
* 最後刪除最後一行的空白行
  • 將 SKILL.md 建立在專案的 .codex/skills/nihonngo 目錄下

Usage

$ Codex "補全 xxx.md"
  • 在 Codex 內下 補全 xxx.md 或 補全，Codex 會自動找到此 Skill 並自動執行

Conclusion

  • 可將 SKILL.md 放在專案目錄下的 .codex/skill 目錄下，此為只有專案使用的 Skill
  • 亦可將 SKILL.md 放在家目錄下的 .codex/skill 目錄下，此為全域的的 Skill，所有專案都能使用
  • 可自行根據需求逐步修改 SKILL.md
