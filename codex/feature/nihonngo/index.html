自動補全日文例句與單字解釋

2026-06-16

在日文學習時，最花時間的是 尋找例句 與 單字解釋，透過 Codex 搭配 AGENTS.md，可自動幫我們 建立例句 與 單字解釋，剩下大量時間。

Version

Codex CLI 0.139.0

Config

AGENTS.md

## 補全

* 將 `意味` 下補全 
* 將 `難度` 下補全
* 將 `情境` 下補全
* 將 `単語` 下補全 (若存在)
* 將 `接続` 下補全 (若存在)
* 將 `例文` 下補全 (若存在)
* 將 `動詞` 下加上例句，並單字解釋 (若存在)
* 將 `い形容詞` 下加上例句，並單字解釋 (若存在)
* 將 `な形容詞` 下加上例句，並單字解釋 (若存在)
* 將 `名詞` 下加上例句，並單字解釋 (若存在)
* 將 `まとめ` 下補全 (若存在)
* 同一個意思的例句只要一句即可
* 不需要 `發布` 到 Hexo
* 最後刪除最後一行的空白行
  • 可自行建立文件大綱，剩下部分由 Codex 自動補全

Conclusion

  • 在 Codex 內輸入 補全 xxx.md，Codex 會自動根據 AGENTS.md 規格補全 Markdown
  • 可自行根據需求逐步修改 AGENTS.md
