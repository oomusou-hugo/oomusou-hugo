CSS 之 ID Selector

2021-04-12

CSS 也可以直接以 ID 描述 Selector，這種方式速度最快，但因為 ID 不能重複，只能套用在單一 Tag。

Version

CSS 3

ID Selector

直接以 id 選擇 CSS 成為紅色。

<template>
  <div id="title">CSS</div>
</template>

<style scoped>
#title {
  font-size: 30px;
  color: #ff0000;
}
</style>

第 2 行

<div id="title">CSS</div>

CSS 的 id 為 title。

第 6 行

#title {
  font-size: 30px;
  color: #ff0000;
}

ID selector 可直接以 #title 選擇 CSS。

Conclusion

  • ID selector 使用 # + id 名稱 描述，只能套用單一 tag，但其 specificity 太高，實務上較少使用
