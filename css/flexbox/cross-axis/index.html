Flexbox 之 Cross Axis 對齊

2019-07-08

row-direction 主要定義 Main Axis 方向，需搭配 justify-content 設定 alignment；若要針對 Cross Axis，則要使用 align-items。

<div> 下有 3 個 <article>，各自內文長短不一。

CSS

style.css

article {
  background: #FFF;
  margin: 5px;
  flex: 1;
}

.box {
  display: flex;
  flex-direction: row;
}

第 7 行

.box {
  display: flex;
  flex-direction: row;
}

display 為 flex，表示使用 flexbox。

flex-direction 為 row，表示 main axis 為 row，cross axis 為 column。

第 1 行

article {
  background: #FFF;
  margin: 5px;
  flex: 1;
}

設定每個 <article> 的 flex 為 1，表示其 flex-grow 為 1，也就是每個 <article> 會以等比例平分 container 剩餘空間。

由於並沒有設定 width，但因為 flex 為 1，所以相當於 3 個 <article> 等寬。

Main Axis vs. Cross Axis

Flexbox 有兩個 axis：

  • Main axis：row-direction 所設定方向
  • Cross axis：與 main axis 垂直方向

Stretch

Container 內 3 個 item 同高。

.box {
  display: flex;
  flex-direction: row;
  align-items: stretch;
}

align-items 預設為 stretch，也就是在 cross axis 方向會自動 stretch 與 container 同高 (寬)。

由於 .box 沒設定 height，其高度是被最長 item 所撐高，較短的 item 會自動 stretch 與其同高

Flex-start

Container 內 3 個 item 在 cross axis 方向皆 靠上對齊。

.box {
  display: flex;
  flex-direction: row;
  align-items: flex-start;
}

align-items 為 flex-start，因為 cross axis 為 column，為 由上至下，相當於 靠上對齊。

Flex-end

Container 內 3 個 item 在 cross axis 方向皆 靠下對齊。

.box {
  display: flex;
  flex-direction: row;
  align-items: flex-end;
}

align-items 為 flex-end，因為 cross axis 為 column，為 由上至下，相當於 靠下對齊。

Center

Container 內 3 個 item 在 cross axis 方向皆 置中對齊。

.box {
  display: flex;
  flex-direction: row;
  align-items: center;
}

align-items 為 center，因為 cross axis 為 column，相當於 垂直置中。

Baseline

Container 內 3 個 item 在 cross axis 方向皆與 第一行 的 baseline 對齊。

.box {
  display: flex;
  flex-direction: row;
  align-items: baseline;
}

align-items 為 baseline，但看起來與 flex-start 一樣。

若將 <header> 拿掉，可明顯看出 baseline 與 flex-start 差異。

Align-self

Container 內第 1 個 item 與第 3 個 item 為 靠上對齊，但第 2 個 item 為 靠下對齊。

.new-york {
  align-self: flex-end;
}

新增 new-york class，align-self 為 flex-end，表示只有套用該 class 的 item 使用獨特 alignment 設定。

align-self 預設為 auto，也就是自動根據 container 的 align-items 設定

Conclusion

  • justify-content 適用於 main axis；align-items 適用於 cross axis
  • align-items 預設為 stretch，會自動 stretch 與 container 同高 (寬)
  • align-items 的 flex-start、flex-end 和 center 與 justify-content 相同，唯套用在 cross axis
  • align-items 的 baseline 會與第一行的 baseline 對齊，實務上較少使用
  • 若有 item 套用獨特的 alignment，就要使用 align-self
