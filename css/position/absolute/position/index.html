使用 Absolute Position 實現定位

2021-03-22

使用 Absolute Position 時搭配 top、bottom、left 與 right 定位時，會往父層尋找具備定位設定的父層。

Version

CSS 3

Flow Layout

紅色 box 位於三層 <div> 內，且每層 <div> 都有其 padding。

<template>
  <div class="parent1">
    <div class="parent2">
      <div class="parent3">
        <div class="box">
          box
        </div>
      </div>
    </div>
  </div>
</template>

<style scoped>
.box {
  width: 100px;
  height: 100px;
  background: #f00;
}

.parent1, .parent2, .parent3 {
  padding: 10px;
}
</style>

第 2 行

<div class="parent1">
  <div class="parent2">
    <div class="parent3">
      <div class="box">
        box
      </div>
    </div>
  </div>
</div>

box 外共有三層 <div>。

14 行

.box {
  width: 100px;
  height: 100px;
  background: #f00;
}

設定 box style：

  • width: 100px：設定 box width
  • height: 100px：設定 box height
  • background: #f00：設定 box 背景顏色

20 行

.parent1, .parent2, .parent3 {
  padding: 10px;
}

設定三層 div style：

  • padding: 10px：設定三層 <div> 的 padding

Absolute Position

紅色 box 從原處浮起成新 layer，第三層 <div> 則內縮只剩下 padding 部分。

<template>
  <div class="parent1">
    <div class="parent2">
      <div class="parent3">
        <div class="box">
          box
        </div>
      </div>
    </div>
  </div>
</template>

<style scoped>
.box {
  width: 100px;
  height: 100px;
  background: #f00;
  position: absolute;
}

.parent1, .parent2, .parent3 {
  padding: 10px;
}
</style>

14 行

.box {
  width: 100px;
  height: 100px;
  background: #f00;
  position: absolute;
}

設定 box style：

  • position: absolute：如同 fixed 一樣，將從原處浮起成立新 layer，因此不佔原本空間，導致 <div> 內縮只剩下 padding

Positin at 3rd div

紅色 box 左側緊貼第三層 <div>。

<template>
  <div class="parent1">
    <div class="parent2">
      <div class="parent3">
        <div class="box">
          box
        </div>
      </div>
    </div>
  </div>
</template>

<style scoped>
.box {
  width: 100px;
  height: 100px;
  background: #f00;
  position: absolute;
  left: 0;
}

.parent1, .parent2, .parent3 {
  padding: 10px;
}

.parent3 {
  position: relative;
}
</style>

14 行

.box {
  width: 100px;
  height: 100px;
  background: #f00;
  position: absolute;
  left: 0;
}

設定 box style：

  • position: absolute：使用 absolute position
  • left: 0：設定 left 為 0，但此時還不知以誰為定位基準，將往父層尋找具備定位設定的父層

若使用 position: fixed，則恆以 window 為基準

26 行

.parent3 {
  position: relative;
}

設定第三層 <div> style：

  • position: relative：設定此層為 relartive 定位

absolute 找到 parent3 而停止，left 將以第三層為標準定位

Position at Window

紅色 box 左側緊貼 browser。

<template>
  <div class="parent1">
    <div class="parent2">
      <div class="parent3">
        <div class="box">
          box
        </div>
      </div>
    </div>
  </div>
</template>

<style scoped>
.box {
  width: 100px;
  height: 100px;
  background: #f00;
  position: absolute;
  left: 0;
}

.parent1, .parent2, .parent3 {
  padding: 10px;
}
</style>

並沒有任何 <div> 設定 position: relative，將定位到最上層的 window。

當父層沒有任何 <div> 使用 position: relative 時，absolute 相當於 fixed

Conclusion

  • 當只有設定 position: absolute 時，將從原處浮起新 layer，此時等效於 fixed
  • 當 position: absolute 搭配 top、bottom、left 或 right 定位時，會往父層尋找具備定位設定的父層
  • 當 position: absolute 搭配 top、bottom、left 或 right 定位時，若在父層都找不到任何有定位的父層，則定位在 window，此時等效於 fixed
