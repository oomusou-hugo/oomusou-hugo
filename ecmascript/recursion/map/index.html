使用 Recursion 實現 map()

2020-03-30

Recursion 為 FP 中重要的一支，亦可使用其實現 map()。

Version

macOS Catalina 10.15.3
VS Code 1.42.1
Quokka 1.0.277
ECMAScript 2015

map()

let data = [1, 2, 3]

let map = f => ([h, ...t]) => !h ? [] : [f(h), ...map(f)(t)]

map(x => x * 2)(data) // ?

Resursion 會將 array 拆分成 head 與 tail，若 head 為 undefined 時為初始值 []，剩下的 tail 將 recursive 呼叫本身的 map() 並使用 ... array spread 展開。

Conclusion

  • 將 array 拆成 head 與 tail 為 recursion 慣用 pattern，然後 tail 會不斷呼叫自己 recursive 解決問題
