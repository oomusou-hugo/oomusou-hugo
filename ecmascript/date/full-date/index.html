將時間拉滿到 23:59:59

2024-06-23

當使用只有 年日 的 日期套件 時，當轉成 Timestamp 時是以 00:00:00，可能造成後端時間區的錯誤，此時可將時間拉滿成 23:59:59。

Version

ECMAScript 2015

setHours() / setMinutes() / setSeconds()

let date = new Date()
date.setHours(23)
date.setMinutes(59)
date.setSeconds(59)
date = date.getTime()
  • setHours()：設定 hour，並不是 n+1，而是直接設定
  • setMinutes()：設定 minute
  • setSeconds()：設定 second

Conclusion

  • set 系列是直接修改 日期 本身，其回傳值為該 日期 的 timestamp
