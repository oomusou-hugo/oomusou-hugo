2026-01-01

本文記錄我的 Firefox 基本設定。

Version

Firefox 146

Installation

$ brew install --cask firefox
  • 使用 Homebrew 安裝 Firefox
  • 使用 Homebrew 安裝 Firefox Developer Edition

Startup

  • Open previous windows and tabs：On
  • Always check if Firefox Developer Edition is your default browser：Off

Interaction

  • Open links in tabs instead of new windows：On
  • Ctrl+Tab cycles through tabs in recently used order：On
  • Ask before closing multiple tabs：Off
  • Ask before quiting with ⌘Q：Off

Website Appearance

  • Website appearance：Automatic

Fonts & Zoom

  • Default font：16
  • Zoom
    • Default zoom
      • 13.6"：120%
      • 15.3"：100%
      • 16"：100%
    • Zoom text only：On

Vertical Tab

  • Vertical tabs：On
  • Show sidebar：On

No Translation

  • browser.translations.enable：false
