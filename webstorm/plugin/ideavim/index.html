使用 IdeaVim 支援 Vim 模式

2025-08-24

Vim 優點在其高效的 鍵盤 控制，可完全不需 滑鼠 或 觸控板 介入。JetBrains 亦官方提供 IdeaVim Plugin，讓我們在 WebStorm 也能使用 Vim 模式。

Version

WebStorm 2025.2
IdeaVim 2.27.2

IdeaVim

  • 安裝 IdeavVim Plugin，為 JetBrains 官方所提供

Hybrid Line Number

Settings -> Editor -> General -> Appearance

  • Show line numbers：Hybrid

  • 同時顯示 絕對 與 相對 行數，方便 Vim 直接跳轉到指定行數

Conclusion

  • WebStorm 無法同時顯示兩行 Line Number，但可透過 Hybrid 同時顯示 絕對 與 相對 行數
