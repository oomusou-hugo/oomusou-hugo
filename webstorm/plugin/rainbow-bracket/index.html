使用 Rainbow Bracket 讓 Block 更明顯

2019-07-11

Rainbow Bracket 是 VS Code 很有名的 Extension，讓 Block 以相同顏色顯示，使 Code 結構更清楚，WebStorm 也可使用。

Version

macOS Mojave 10.14.5
WebStorm 2019.1.3
Quokka 1.0.232
Rainbow Bracket 5.20

Rainbow Bracket

在 Marketplace 搜尋 Rainbow Bracket 安裝。

HTML 同一個 tag 使用相同顏色表示。

JavaScript 同一個 block 也使用相同顏色表示。

Conclusion

  • Rainbow Bracket 是很實用的 plugin，推薦大家使用
