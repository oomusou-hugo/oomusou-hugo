使用 Alpine.js Support Plugin 支援 Alpine

2024-01-26

WebStorm 目前還沒有支援 Alpine，但可按裝 3rd Party 的 Alpine.js Support Plugin 支援 Alpine。

Version

WebStorm 2023.3.2
Alpine.js Support plugin 0.6.4

Alpine.js Support Plugin

Settings -> Plugins

  • 安裝 Alpine.js Support plugin

  • 支援 Alpine 的語法提示

Conclusion

  • 目前 Alpine.js Support plugin 支援不算完整，但最少可避免 WebStorm 誤認 Alpine 語法為錯誤
