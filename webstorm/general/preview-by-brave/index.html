Previewing HTML by Brave

2022-07-29

Since Chrome is bad, we often use Brave to replace Chrome for development. We can still use Brave to preview on WebStorm.

Version

WebStorm 2022.2

Preferences -> Tools -> Web Browsers and Preview

  • Chrome：Brave Browser

Just change Chrome to Brave Browser, we don’t have to specify the real path of Brave

Conclusion

  • WebStorm still works well on WebStorm
