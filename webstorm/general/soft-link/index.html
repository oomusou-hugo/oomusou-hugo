如何以更簡短 Command 啟動 WebStorm ?

2019-08-02

WebStorm 目前也可在 CLI 以 webstorm Command 直接開啟目前目錄的 Project，但 webstorm 稍嫌冗長，該如何改以 ws 啟動呢 ?

Version

macOS Mojave 10.14.5
WebStorm 2019.2

WebStorm

預設在 /usr/local/bin 下已經有 webstorm shell script，因此我們可在 CLI 使用 webstorm 開啟目前目錄下的 project。

但 webstorm 稍嫌冗長，能否改以 ws 取代呢 ?

直覺會想將 webstorm rename 成 ws，這樣雖然可行，但將來日後 WebStorm 升級改版，Toolbox 又會建立一個新的 webstorm，ws 將無法使用。

Soft Link

$ ln -s webstorm ws

在 /usr/local/bin 下輸入 ln 建立 ws soft link 指向 webstorm。

ln source_file target_file
建立 linke file

  • -s：s ymbolic link
  • source_file：原本檔案
  • target_file：新的 link file

將多了 ws 指向 webstorm。

如此就可使用 ws . 將目前目錄的 project 以 WebStorm 開啟，且將來 Toolbox 更新 WebStorm 也不用擔心。

Conclusion

  • ln -s 是很好用的小技巧，可以讓很多工具以更簡短方式執行
