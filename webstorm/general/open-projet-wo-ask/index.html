如何在 WebStorm 自動切換 Project ?

2020-07-29

以 JetBrains ToolBox 快速切換 Project 時，WebStorm 預設會 詢問 是否開啟，可加以設定為 不詢問 自動開啟 Project。

Version

macOS Catalina 10.15.6
WebStorm 2020.2

Open Project with Ask

當使用 toolbox 切換 project 時，若 WebStorm 已開啟，會詢問從 Attach、New Windows、This Window 三種模式選擇其一開啟。

Preferences -> Appearance & Behavior -> System Settings

  • Project -> Reopen projects on startup -> Current windows：true

如此就會自動切換 project 而不會另外詢問。
