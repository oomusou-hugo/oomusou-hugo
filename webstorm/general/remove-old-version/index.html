如何移除舊版 WebStorm ?

2019-06-14

使用 Toolbox 更新 WebStorm 時，預設並不會移除舊版 WebStorm，JetBrains 本意是讓你能快速 Rollback 到之前版本，但這也無形浪費 SSD 空間，其實也可以移除舊版 WebStorm，並設定只留下最新版本。

Version

macOS Mojave 10.4.5
WebStorm 2019.1.3

Toolbox

在 Toolbox 的 settings 下，Keep only the latest version 預設為 disable，可將此選項 enable，則未來僅管 WebStorm 更新，將只留下最新版本。

此外，可按下 Remove the older versions 移除舊版 WebStorm，將回復不少 SSD 空間。

Conclusion

  • 很多人不知道 Toolbox 會留下舊版 WebStorm 造成 SSD 空間浪費，透過此設定將只留下最新版本，並將舊版 WebStorm 全部移除
