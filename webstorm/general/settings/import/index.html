將 ZIP 檔案匯入設定

2026-05-18

將 WebStorm 設定匯出成 ZIP 檔之後，可再其他電腦加以匯入，不必再由 GUI 慢慢設定。

Version

WebStorm 2026.1.2

Import Settings

File -> Manage IDE Settings -> Export Settings

  • 選擇要匯入的 ZIP 檔

  • 可選擇匯出部分，或 Select All 全部匯出
  • 可選擇匯出位置

  • 匯入成功，重新啟動 WebStorm 套用新設定

Conclusion

  • WebStorm 所匯出的其實是多個 XML 檔，最後再以 ZIP 壓縮成一個檔一次匯入
