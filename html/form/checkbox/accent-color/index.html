改變 CheckBox 的背景顏色

2023-08-05

可使用 accent-color 直接改變 CheckBox 的背景顏色。

Version

HTML 5

accent-color

CheckBox 背景顏色已改成 藍色。

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Title</title>
    <style>
      .my-checkbox {
        accent-color: blue;
      }
    </style>
  </head>
  <body>
    <label><input class="my-checkbox" type="checkbox" />Swimming</label>
  </body>
</html>

第 7 行

.my-checkbox {
  accent-color: blue;
}
  • accent-color：改變背景顏色
