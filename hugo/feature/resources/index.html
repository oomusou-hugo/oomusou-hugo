resources 目錄簡介

2024-04-08

resources 目錄並不是一開始就有，而是當下了 hugo server 才會建立，我應該手動刪除此目錄嗎？

Version

Hugo 0.90

Resources Folder

  • 當使用 hugo server 指令後，resources 將被建立在專案的根目錄

  • 主題的圖片、JavaScript 或 CSS 將被最佳化並且 cache 在 resources 目錄下

Conclusion

  • 不建議將 resources 刪除，重建這些 cache 需要一些處理時間，但不必將 resources 目錄放進 Git 管理
