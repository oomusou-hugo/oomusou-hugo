我的 Zen 基本設定

2026-01-01

本文記錄 Zen 的安裝與基本設定。

Version

Zen Browser 1.17

Homebrew

$ brew install --cask zen
  • 使用 Homebrew 安裝 Zen

Layout

  • Tabs on the right

Startup

  • Open previous windows and tabs：On
  • Continue where you left off：On
  • Always check if Zen is your default browser：Off

Tabs

  • Ctrl + Tab cycles through tabs in recently used order：On
  • Open links in tabs instead of new windows：On
  • Open links from apps next to your active tab：On
  • Ask before closing multiple tabs：Off
  • Ask before quitting with ⌘Q：Off
  • Enable Container Tabs：On

Language and Appearance

  • Automatic

Fonts & Zoom

  • Default font：16
  • Zoom
    • Default zoom
      • 13.6"：120%
      • 15.3"：100%
      • 16"：100%
    • Zoom text only：Off

Look and Feel

  • Layout：Only Sidebar
  • Vertical Tabs
    • Show New Tab Button on Tab List：On
      • Move the new tab button to the top：On

No translation

  • about:config
  • browser.translations.enable：false

Sync

  • Firefox Account

Extension

  • Dark Reader (Pin)
  • Vimium
  • Pesticide for Firefox

Tabs

  • MyNippon (100%)
  • MyHugo (100%)
  • MyTech (100%)
  • Gemini (150%)
  • ChatGPT (150%)
  • Grok (150%)

Shortcut

  • ⌘ S：切換 Side Bar
  • ⌘ T：新 Tab
  • ⌘ W：關閉 Tab
